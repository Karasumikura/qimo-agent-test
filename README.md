# 期末重点智能体 MVP

这是一个本地运行的课程复习智能体原型，用来处理你有权访问的课程资料：

- 上传课堂视频、音频、字幕、文字稿、PDF、DOCX、笔记和往年题。
- 从视频/音频中提取 16kHz 音频。
- 可选接入 `faster-whisper` 自动转写。
- 自动导入学在吉大页面中可正常访问的 `mp4` / `m3u8` 视频资源。
- 根据老师提示词、课堂证据和往年题命中生成期末重点报告。
- 可选接入 OpenAI-compatible LLM，对老师暗示、命题方向和复习动作做语义归纳。
- 预留“学在吉大”授权连接器接口，不绕过登录、验证码、DRM 或平台限制。

## 启动

```powershell
python -m venv .venv
.\.venv\Scripts\python -m pip install -r requirements.txt
.\.venv\Scripts\python -m uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

打开：

```text
http://127.0.0.1:8000
```

## 使用流程

1. 输入课程名。
2. 选择一种资料导入方式：手动上传、粘贴文本、粘贴视频地址，或使用浏览器书签从学在吉大视频页导入。
3. 如果导入视频/音频，系统会先用 `ffmpeg` 提取音频。
4. 如果启用了 Whisper，会自动转写；否则会保留音频并提示补充文字稿。
5. 点击“生成报告”，或等待远程导入完成后的自动分析。
6. 下载 Markdown 报告。

## LLM 语义分析

前端可以直接配置 OpenAI-compatible 接口：

- `baseURL`：例如 `https://api.openai.com/v1`、DeepSeek/硅基流动/vLLM/Ollama 的兼容地址。
- `APIKEY`：只保存在当前浏览器会话里，并随“生成报告”请求发送给本地后端，不写入 SQLite、报告或日志。
- `模型`：例如 `gpt-4o-mini`、`deepseek-chat`，按你所填服务商支持的模型名填写。
- `温度`：默认 `0.2`。

不开启 LLM 时，系统仍会使用本地规则分析。开启 LLM 后，后端调用 `${baseURL}/chat/completions`，把课堂文字摘录、规则分析候选知识点和往年题证据发给模型，要求模型返回 JSON，再合并进报告。

远程视频导入完成后的后台自动分析不会持久保存 APIKEY。需要 LLM 增强时，建议等资料处理完成后在前端手动点击“生成报告”。

## 学在吉大自动导入

支持两种方式：

- **视频地址导入**：把你在授权页面中拿到的 `mp4` 或 `m3u8` 地址粘贴到“视频地址”输入框。
- **浏览器书签导入**：把页面里的“导入到期末智能体”拖到浏览器书签栏。登录学在吉大，打开并播放目标视频后点击该书签，系统会把页面暴露的候选视频地址提交到本地服务，后台下载并进入分析流程。

导入说明：

- 用户登录仍在浏览器中手动完成，本地服务不会读取浏览器 Cookie、密码或登录数据。
- 如果页面暴露的是临时签名 URL 或无 `.mp4` 后缀的视频请求，书签会作为候选地址提交给后端逐个尝试。
- 如果页面已有“转文字/字幕/文稿”，书签会同步提交这段文字；即使视频下载失败，也可以先用文字稿分析。
- 更新书签逻辑后，需要从本地页面重新把“导入到期末智能体”拖到书签栏，旧书签不会自动更新。
- 如果视频必须依赖浏览器 Cookie、加密播放器、DRM 或分片加密，系统不会绕过这些限制。
- 为避免本地网络探测，默认禁止导入 `127.0.0.1`、内网地址和私有 IP 视频链接。

## 自动转写

默认不强制安装 Whisper，避免首次安装过重。需要自动转写时：

```powershell
.\.venv\Scripts\python -m pip install faster-whisper
$env:ENABLE_WHISPER="1"
$env:WHISPER_MODEL="small"
.\.venv\Scripts\python -m uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

如有 NVIDIA GPU，可按本机 CUDA 环境调整 `WHISPER_DEVICE` 和 `WHISPER_COMPUTE_TYPE`。

## 合规边界

这个项目只处理公开课程、个人授权访问课程、老师授权资料或你手动上传的学习资料。不要用于绕过平台访问控制、验证码、DRM、反下载策略，或传播未授权课程内容。
